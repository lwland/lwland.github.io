Github Pages + Hexo搭建个人博客

  1. 获取Github Pages
    1. 首先有个github账号
    2. 新建一个repo，名称一定是your_username.github.io这样的格式。
    3. 在刚创建的repo，Setting-Automatic Page Generator 选个主题
      比如你的用户名为zhangsan，Repository name里面就填上zhangsan
  2. Hexo搭建个人博客
    1. nodejs下载，地址：ttp://nodejs.cn/download/。node -v 查看是否安装成功
    2. 安装hexo，npm install hexo-cli -g ，没有权限的话使用sudo命令
    3. 创建博客目录，cd到想搭建博客的路径
    4. 初始化，hexo init，会初始化博客的目录，所以，执行这个命令时，想博客的目录下执行
    5. 安装依赖包， hexo install
    6. 生成，hexo g
      • 目录
    7. 运行，hexo s
    8. hexo 常用命令
      1. hexo new “postName” #新建文章
      2. hexo new page “pageName” #新建页面
      3. hexo generate #生成静态页面至public目录
      4. hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
      5. hexo deploy #将.deploy目录部署到GitHub
  3. Hexo 部署到github
    1. npm install hexo-deployer-git –save
    2. hexo d来部署
